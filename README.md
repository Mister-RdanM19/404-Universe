#  404-Universe: The Ultimate Collection

![Language](https://img.shields.io/badge/Language-HTML%2FCSS%2FJS-orange)
![Developer](https://img.shields.io/badge/Developer-Mr.Rm19-blue)

**404-Universe** adalah sebuah galeri interaktif yang berisi 19 desain halaman 404 (Page Not Found) unik dengan berbagai tema—mulai dari *Retro Gaming*, *Cyberpunk*, hingga *Developer Tools*. Project ini dibuat untuk memberikan pengalaman visual yang berbeda bagi pengunjung website saat mereka tersesat di URL yang salah.

##  Fitur Utama
* **19 Tema Variatif:** Koleksi lengkap dari kategori Space, Gaming, Tech, hingga Minimalis.
* **Dashboard Interaktif:** Dilengkapi dengan `index.html` bergaya *Glassmorphism* untuk navigasi antar tema.
* **Responsive Design:** Optimal untuk dibuka di perangkat desktop maupun mobile.
* **Modern Tech Stack:** Menggunakan HTML5, CSS3, dan Vanilla JavaScript.
